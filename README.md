# gatsby-plugin-shopify-buy

## Install

```
npm install --save gatsby-plugin-shopify-buy
```

## Usage

Add the plugin to your `gatsby-config.js`:

```js
// gatsby-config.js

module.exports = {
  plugins: [
    {
      resolve: `gatsby-plugin-modal-routing`,
      options: {
        shopName: `the-name-of-your-shopify-store`,
        accessToken: `xxxxx`,
      }
    }
  ]
];
```
